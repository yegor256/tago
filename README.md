First, install it:

```bash
gem install tago
```

Here is how you use it:

```ruby
start = Time.now
# something long
puts "It took #{start.ago} to do it"
```

It's also possible to convert Float seconds to text:

```ruby
s = Time.now - start
puts "It took #{s.seconds}"
```

The gem basically extends the `Float` and `Time` classes with new methods.
